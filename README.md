# 🏭 Manufacturing Quality Control Dashboard

## 📌 Project Overview

This project is an end-to-end Manufacturing Quality Control Dashboard developed in Power BI.

The dashboard helps manufacturing companies monitor production performance, product quality, operational efficiency, supplier performance, and energy utilization.

## 🎯 Business Problem

Manufacturing companies often struggle to monitor:

- Product defects
- Production efficiency
- Machine downtime
- Maintenance activities
- Supplier quality
- Energy consumption
- Inventory performance

This dashboard provides actionable insights to support data-driven decision-making.


## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Star Schema Data Modeling
- Microsoft Excel


## 📊 Dashboard Pages

### 1️⃣ Executive Overview

- Production KPIs

- Factory Performance

### 2️⃣ Quality Analysis

- Product Quality

### 3️⃣ Production & Operations

- Production Monitoring
- Worker Productivity
- Maintenance Analysis

### 4️⃣ Supplier & Energy Analysis

- Supplier Performance
- Delivery Delay
- Energy Consumption
- Inventory Turnover
- Stockout Analysis

## 📈 Key KPIs

- Total Production Volume

- Defect Rate %

- Supplier Quality
